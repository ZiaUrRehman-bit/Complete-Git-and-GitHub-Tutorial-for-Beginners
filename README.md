## Git
Version control system is a tool that helps to track changes in code<br>
Git is a Version Control System.
+ Popular
+ Free & Open Source
+ Fast & Scalable
  
Git is mainly used for two things.<br>
+ To track the history of changes
+ To collaborate (work in a team)
